# Todo App

Простое fullstack-приложение для управления задачами.  

## Live Demo

https://ereminaolesya.github.io/todo-app/

## Stack

- Frontend: JavaScript, HTML, CSS  
- Backend: ASP.NET Core Web API с Entity Framework Core  

## Функционал

- Добавление задач  
- Редактирование текста задачи  
- Отметка выполнения  
- Удаление задачи  
- Импорт/экспорт JSON
